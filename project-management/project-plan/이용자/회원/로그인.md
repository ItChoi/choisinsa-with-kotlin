# 로그인

## 로그인 방식
- 로그인 방식은 두 가지가 존재한다.
  1. id/pw 입력 로그인
  2. sns 로그인 
     - 애플
     - 카카오
     - 구글

---

## 로그인 절차
- id/pw 입력 로그인
  - 입력 정보 일치
    - 메인 화면 이동
  - 입력 정보 불일치
    - 입력 정보 불일치 메시지 (어떤 입력 정보가 실패했는지 자세히 제공 X -> "입력한 정보가 일치하지 않습니다.")
- sns 로그인
  - sns 인증 성공
    - 회원 O
      - 메인 화면 이동
    - 당사 X
      - 회원가입 페이지 이동 -> [회원가입 절차 - sns 회원가입](회원가입.md#회원가입-절차)
